# Learning Roadmap

1. FastAPI basics: routes, dependency injection, validation, and app structure.
2. REST API design: resources, status codes, pagination, filtering, and error responses.
3. PostgreSQL modelling: relational design, constraints, indexes, and ownership boundaries.
4. SQLAlchemy: ORM models, sessions, queries, and repository/service patterns.
5. Alembic migrations: versioned schema changes and migration review.
6. Authentication: password hashing, JWTs, current-user dependencies, and authorization.
7. External API integration: provider abstraction, error handling, retries, and timeouts.
8. Caching: cached dictionary data, invalidation rules, and future Redis usage.
9. Testing: unit tests, API tests, database tests, fixtures, and test isolation.
10. Background jobs: scheduled review work and asynchronous processing.
11. Docker: reproducible local development with API and PostgreSQL containers.
12. CI/CD: automated tests, linting, and deployment pipelines.
13. Observability: structured logging, metrics, tracing, and operational dashboards.
14. Deployment: environment configuration, managed database, secrets, and release process.
15. AI-assisted features later: generated explanations, examples, and learning support behind clear service boundaries.
