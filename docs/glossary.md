# Glossary

## Word

A shared dictionary entry, such as `serendipity`, stored independently from any user's glossary.

## Definition

A meaning associated with a word, usually including part of speech and source.

## User Word

A word saved by a specific user into their personal glossary.

## Review

The scheduling state used to decide when a user should practice a saved word again.

## Spaced Repetition

A learning technique that schedules reviews at increasing intervals based on how well the user remembers an item.

## Dictionary Provider

An external or internal source of dictionary data. The first planned provider is Free Dictionary API.

## Global Dictionary Cache

Stored dictionary data shared by all users to avoid repeated external lookups for the same word.

## Learning Status

A user-specific state describing progress for a saved word, such as new, learning, reviewing, or mastered.
